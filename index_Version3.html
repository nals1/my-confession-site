Why this setup
  • Viewable by anyone with the link (no sign-in required).
  • Only signed-in users can send/edit/delete their messages.
  • If you enable encryption, the server stores ciphertext only; key is inside the link fragment and never sent to the server.
  • Messages cannot be listed by other people.
